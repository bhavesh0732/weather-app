# weather-app

A simple and responsive Weather Application built using HTML, CSS, and JavaScript. It fetches real-time weather data using a public API and displays it in a clean UI.
 Features

- 🔍 Search weather by city name  
- 🌡️ Displays temperature, humidity, and weather condition  
- 📡 Real-time data using API  
- ⚡ Fast and responsive UI  
- ❌ Error handling for invalid city names
- Technologies Used

- HTML5  
- CSS3  
- JavaScript (Vanilla JS)  
- REST API (OpenWeather API)
